# Revision History

## English

### v1.0.0 - 2026-06-11

- Initial public release of `Intelligence Forge`.
- Defines the installable skill name `intelligence-forge` and display name `Intelligence Forge`.
- Adds Stage 0 domain onboarding before private corpus induction.
- Adds evidence-backed claim lifecycle, counterexample pressure, blind validation, and user-feedback regression.
- Adds promotion gates for `domain_core` and `validated_general` to reduce false generalization.
- Preserves under-proven but useful knowledge as contextual rules, overlays, special cases, unknowns, or `Cannot infer` boundaries.
- Adds scripts for manifesting corpora, shallow profiling, candidate extraction, coverage scoring, counterexample search, deep-read planning, validation, question ledger rendering, feedback capture, and iteration reporting.
- Release remains unlicensed unless the repository owner later adds a license.

## 中文
