# Deeply Reader

Evidence-grounded deep reading skill for book-like PDF, DOCX, EPUB, Markdown, and TXT files, producing useful intelligence, cognitive maps, and actionable reading notes.

## What It Helps With

Use this branch when a long book-like document needs more than a shallow summary and the result should preserve evidence-aware insights.

## Included Skills

- `deeply-reader/`: Create evidence-grounded deep reading HTML reports from whole-book files (PDF/DOCX/EPUB/MD/TXT). Use for book intelligence, cognitive maps, method extraction, actionable notes, and work/life reflection; not document editing, OCR, conversion, slides, or shallow mind maps.

## Installation

Open the `Deeply-Reader-Skill` branch in `Willis1214/Skills` and copy the needed skill folder into your Codex skills directory:

```text
~/.codex/skills/<skill-folder>
```

For this branch:

- Copy `deeply-reader/` into your Codex skills directory to install `deeply-reader`.
